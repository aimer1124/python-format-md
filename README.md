# python-format-md
为什么做这个：需要把原来的文章内容从[Hexo](hexo.io/)转移至[Hugo](https://gohugo.io/)，但两者在MD文件头部处理不一致。

- Hexo格式

```
title: 'Http-Header'
tags:
  - API测试
  - SuperTest
categories:
  - Tool
date: 2016-02-29 20:14:00
thumbnail: /img/a.png
---
```

- Hugo格式 

```
---
title: 'Http-Header'
tags:
  - API测试
  - SuperTest
categories:
  - Tool
date: 2016-02-29 20:14:00
---
```
